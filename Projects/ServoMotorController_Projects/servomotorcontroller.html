Servo Motor Controller

Objective

The goal of this project was to create a circuit capable of controlling the speed of a servo motor. A motor-driven disc with a slit generates a clock pulse each time the slit passes under an LED, allowing light to reach a photoresistor. These pulses are fed into a counter, which measures the number of pulses between reset signals. The counter output is then converted from digital to analog using an R-2R resistor ladder network and compared to the motorâ€™s voltage. The resulting error signal is amplified and used to adjust the motor speed by regulating the current.
